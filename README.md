# HarmonyHub

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/hlokomani/hobbystore
   ```

2. Navigate to the project directory:
   ```
   cd harmonyhub-auth
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```
